## Shortcuts

|Keys|Result|
|---|---|
|Ctrl + Enter| Run (selected) code|
|Ctrl + Shift + Enter|Run all code|
|Ctrl + Shift + M|%>%|
|Alt + -	|<-|
|Ctrl + Shift + C|(Un)comment block|
|Ctrl + Shift + /|Reflow comment|
|Alt + Shift + K|Show keyboard shortcuts|
|Alt + o|Fold all code|
|Alt + Shift + o|Unfold all code|
